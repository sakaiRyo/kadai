========================================================================
    コンソール アプリケーション: ConsoleApplication1 プロジェクトの概要
========================================================================
code　ソース類格納先
data  変換元データ入力元
out   変換後データ出力先

実装状況
指定されたディレクトリ以下のディレクトリを探索し、それ以下のディレクトリを洗い出す。
入っていたディレクトリ分のスレッドを作成し、さらに画像データの探索を行う。
データの変換を行う。
画像の変換部分は実装途中。
